[[at_input_type]]
=== @InputType

Controls the mapping of the annotated Java type to a GraphQL input object type.

.Parameters
[cols="1,1"]
|===
|*name*|The name the corresponding input object type should have in the GraphQL schema
|*description*|The description of the corresponding input object type
|===

.Example
[source,java,numbered]
----
@Type(name = "Starship", description = "A starship from Starwars")
@InputType(name = "StarshipInput", description = "Input object for a starship")
public class Starship {
   private String id;
   private String name;
   private float length;

   //  Getters, setters...
}
----
[source,java,numbered]
----
public class StarshipService {
  @Mutation(name = "saveStarship", description = "Save a new starship")
  public Starship save(Starship starship) {
     // ...
  }
}
----

Resulting schema SDL:

[source,json,numbered]
----
# Input object for a starship
input Starship {
   id: String
   name: String
   length: Float
}
----

[source,json,numbered]
----
type Mutation {
  # Creates and saves a new starship
  saveStarship(starship: StarshipInput): Starship
}
----
