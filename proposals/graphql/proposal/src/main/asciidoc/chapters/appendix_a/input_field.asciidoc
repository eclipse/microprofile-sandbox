[[at_input_field]]
=== @InputField

Controls the mapping of the property to an input object field. These are distinct from fields on an output object.

.Parameters
[cols="1,1"]
|===
|*name*|The name of the corresponding input object field should have in the GraphQL schema
|*description*|The description of the corresponding input object field
|===

.Example
[source,java,numbered]
----
@Type(name = "Starship", description = "A starship from Starwars")
@InputType(name = "StarshipInput", description = "Input object for a starship")
public class Starship {
   @InputField(name = "uuid", description = "uuid of a new Starship")
   private String id;
   private String name;
   private float length;

   //  Getters, setters...
}
----

Resulting schema SDL:

[source,json,numbered]
----
# Input object for a starship
input Starship {
   # uuid of a new starship
   uuid: String
   name: String
   length: Float
}

# A starship from Starwars
type Starship {
   id: String
   name: String
   length: Float
}
----
